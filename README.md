# Big Data Analysis - YouTube Streamer Stats

A comprehensive big data analysis project that processes and analyzes YouTube streamer statistics using Jupyter notebooks.

## Overview

This project leverages big data tools and techniques to collect, process, and analyze large-scale YouTube streamer performance metrics. The analysis provides insights into streaming trends, viewer engagement, and channel growth patterns.

## Features

- **Data Collection** - Aggregates YouTube streamer data at scale
- **Data Processing** - Cleans and transforms raw data for analysis
- **Statistical Analysis** - Performs comprehensive statistical evaluations
- **Visualization** - Creates insightful charts and graphs
- **Jupyter Notebooks** - Interactive analysis and documentation

## Tech Stack

- **Jupyter Notebooks** - Interactive data analysis and visualization
- **Python** - Primary programming language
- **Pandas** - Data manipulation and analysis
- **Big Data Tools** - For processing large datasets

## Getting Started

### Prerequisites

- Python 3.7+
- Jupyter Notebook
- Required Python packages (see requirements.txt)

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Launch Jupyter:
   ```bash
   jupyter notebook
   ```

## Usage

Open the Jupyter notebooks in the project directory to explore the analysis. Each notebook contains detailed explanations and visualizations of the YouTube streamer statistics.
